**3BL – Mission 02: Jake and Jane
Greybox to Vertical Slice Development**

This repository contains the active development build of Jake and Jane, the second story mission from the original AAA game IP, 3BL. The project is currently in the Prototype / Vertical Slice stage of development.

Jake and Jane serves as the foundational greybox implementation of 3BL’s core systems and encounter design. The objective is to progressively refine this greybox into a polished vertical slice that represents final production standards in gameplay mechanics, AI behavior, combat structure, and technical architecture.

Development began in January 2026, with the vertical slice targeted for completion by December 2026. This milestone is intended to validate execution capability, performance stability, and scalable system design before transitioning into full-scale production.


**Project Goals**
**1. Deliver a Production-Quality Vertical Slice**
Transform the existing greybox mission into a polished vertical slice that accurately reflects intended gameplay pacing, encounter tension, and technical quality benchmarks.

**2. Validate Core Combat Systems**
Establish and refine the foundational gameplay loop of 3BL, including:
Tactical weapon acquisition
Precision-based shooting mechanics
Structured duel system (6-round limit, 3-hit victory condition)
Clear mission-state progression and outcome logic
The focus is on clarity, balance, and measurable system reliability.

**3. Develop Rule-Based AI Combat Architecture**
Implement intelligent AI behavior using Unreal Engine systems, enabling:
Player detection and evaluation
Tactical aiming and controlled firing
Ammo-aware decision-making
Defensive positioning and cover behavior
The AI operates within defined combat constraints rather than scripted sequences.

**4. Establish Scalable Technical Structure**
Build modular, production-ready systems that support future expansion. Core architecture is designed to scale beyond this mission without restructuring foundational gameplay logic.

**5. Benchmark Performance and Stability**
Use the vertical slice to validate:
Real-time rendering performance
AI system efficiency
Combat responsiveness
Integration between gameplay and cinematic transitions
This ensures that technical decisions are aligned with long-term AAA production standards.

**6. Create a Demonstrable Capability Milestone**
Deliver a publicly presentable vertical slice by December 2026 that showcases technical competence, gameplay depth, and readiness for scaled development of the 3BL IP.
